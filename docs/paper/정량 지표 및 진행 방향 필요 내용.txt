3. 연구 방법 및 내용 (확장 초안)

본 장은 “의(義): AI와 그려나갈, 공존하는 사회”라는 공모전 주제에 맞추어, 정서적 취약계층을 보호하고 사회적 상호공존을 촉진하기 위해 설계된 Red Heart AI의 방법론을 세밀하게 기술한다. 구현은 SLM(Small Language Model) 기반의 직접 계산 철학을 따르며, (1) 다차원 감정 분석(Emotion + DSP), (2) 정량 윤리 판단(Advanced Bentham Calculator), (3) 반사실적 후회 분석(Regret), (4) **정보이론 기반 인과 분해(SURD)**를 통합한다. 이 장에서는 데이터·모델·학습·추론·안전 전 과정을 말로 풀어 단계적으로 설명하며, **검증과 결과(4장)**는 학습 완료 후에 별도로 제시한다.

3.1 문제 정의와 설계 원칙
3.1.1 문제 정의

입력(텍스트 중심, 멀티모달 확장 가능)으로부터 감정 상태를 정밀 추정하고, 윤리적 효과를 정량 평가하며, 대안 선택의 결과를 반사실적으로 시뮬레이션하고, 그 과정의 정보 흐름과 인과 구조를 해석한다.

목표는 “취약계층의 최소 피해와 존엄 보장”이며, 단순한 최대 효용이 아니라 위험 회피·인지 부담 완화·정서 보호를 우선한다.

3.1.2 설계 원칙

직접 계산·결정론: 생성형 임의성 대신 명시적 계산 절차와 추적 가능한 근거를 우선한다.

설명가능성: 각 점수의 구성 요소, 가중, 불확실성, 대안 시나리오를 문장으로 설명한다.

모듈 상호검증: 감정·윤리·후회·인과 모듈이 서로의 출력을 교차 점검하여 할루시네이션을 억제한다.

사회·문화 맥락: 한국 문화(정/한/체면)와 공동체 규범을 반영하는 보정 절차를 포함한다.

8GB 환경 최적화: 경량 백본 공유, 동적 경로 선택, 혼합정밀로 실시간성과 안정성을 확보한다.

3.2 시스템 전반 구조
3.2.1 공유 백본과 전용 헤드

공유 백본: 텍스트 임베딩, 시간·문화 임베딩, 상황 특징을 결합해 공통 표현을 만든다. 백본은 Cross-Attention으로 각 헤드(감정/윤리/후회/SURD)에 필요한 정보를 공급한다.

전용 헤드: 각 모듈은 목적에 맞춘 얕은 계층(설명가능성 우선)을 사용한다. 복잡한 계산은 **외부 계산 블록(규칙·DSP·정보이론)**에서 수행하고, 헤드는 결과를 정규화·집계한다.

고급 분석 계층: 감정 MoE, 벤담 6층 가중, 후회 3-View+베이지안, SURD PID+GNN이 평행 배치된다.

3.2.2 실행 경로(요약)

입력 정규화 → 2) 백본 표현 → 3) 감정 모듈(시계열 안정화 포함) → 4) 윤리 모듈(10D 변수·6층 가중·3-View) → 5) 후회 모듈(반사실적 시나리오, 베이지안 통합) → 6) SURD(정보 분해·인과 해석) → 7) 삼각 회로 통합(가중 결합, 취약성 임계·감독) → 8) 설명 템플릿 생성.

3.3 다차원 감정 분석 (Emotion + DSP)
3.3.1 목표와 출력 형식

목표: 텍스트로 드러난 정서를 다차원(기본 7정서 + 하위·미세 감정 + 한국 특화 정서)으로 추정하고, 시간적 궤적과 신뢰도·불확실성을 산출한다.

출력: (a) 정서 라벨 분포와 최빈 라벨, (b) 강도·지속·복합성, (c) 문화 정서(정/한/체면), (d) Valence–Arousal 좌표와 시계열, (e) 신뢰도와 품질 플래그(반어법·모호·충돌 등).

3.3.2 입력 전처리와 특징화

언어 전처리: 문장 분할, 구문 단서(부정, 강조, 완곡), 대용어 해소, 담화 역할(요청/고백/비난) 태깅.

정서 어휘·패턴 특징: 정서 키워드 밀도, 감탄사·의성어, 표정·상징 은유(“가슴이 무겁다” 등) 추출.

상황·문화 특징: 관계성 지표(나/너/우리), 체면 관련 단어, 유교적 규범 단서, 사회적 맥락(공적/사적) 표지.

3.3.3 EmotionDSP: 신호처리 결합

필터 스택: 언어 신호의 변동을 저역통과(급변 완화), 고역통과(미세 변화 강조), 대역통과(특정 감정 밴드)로 분해해 가중 재합성한다. 이렇게 얻은 정제 감정 신호는 문장 단위의 과도응답을 줄이고, 감정 전환의 모서리를 부드럽게 한다.

칼만 동역학: 상태는 Valence, Arousal, Dominance와 그 1차 변화율로 정의한다. 예측-보정 루프로 연속 프레임의 감정 궤적을 추정하며, 측정 잡음(언어 모호성)과 모형 잡음(빠른 심경 변화)을 분리 관리한다.

효과: 순간적 과대평가를 억제하고, 진성 변화(지속·경향)를 드러내어 후속 윤리·후회 모듈의 안정적 입력을 보장한다.

3.3.4 계층적 감정 해상도와 문화 맥락

3단계 해상도: (L1) 7 기본정서 → (L2) 28 하위정서 → (L3) 112 미세 뉘앙스. 상위→하위로 내려갈수록 감정 혼합과 양가감정을 별도 차원으로 표기한다.

문화 특화: 한국 고유 정서인 정/한/체면을 별도 축으로 예측한다. 체면이 강한 맥락에서는 표면 긍정과 내면 부정의 이중 신호가 흔하므로, 담화 기능(사과·완곡)과의 결합 규칙을 둔다.

3.3.5 역설·아이러니·문체 처리

반어법 탐지: 긍정 어휘 + 부정 문맥(이모티콘·따옴표·말끝 흐림)을 규칙+확률로 결합해 반전 플래그를 부여한다.

강조/약화: 반복, 과장, 축약 표현을 스케일링 계수로 반영한다.

3.3.6 개인화의 초석(옵션)

대화 히스토리를 메모리 뱅크에 축적해 사용자별 기준선을 형성하고, 동일 사용자에서 반복되는 감정 패턴은 사전 확률로 반영한다(개인정보는 3.12에 따라 최소 수집·익명화).

3.3.7 불확실성·품질 지표

라벨 엔트로피(분포 확산), 시계열 일관성(프레임 간 변동), 충돌 지수(긍/부 묶음 동시 활성)로 신뢰도를 산출한다. 신뢰도가 낮으면 윤리/후회 모듈의 보수적 경로를 선택한다.

3.4 정량 윤리 판단 (Advanced Bentham Calculator)
3.4.1 평가 변수의 조작적 정의(10차원)

강도: 당사자·타자의 쾌/고통 강도. 언어 단서의 감정 강도, 상황의 물리·경제 규모로 포착.

지속: 효과의 지속 기간. 문맥 내 시간 단서(한 번/장기)와 사례 유사도 기반 추정.

확실성: 발생 확률. 근거 개수·질·출처 일치도로 산정.

근접성: 시간·공간·관계 거리. 당사자성(본인/지인/타인)과 발생 시점(즉시/지연).

다산성: 추가 쾌락·고통의 연쇄 가능성. 2·3차 파급을 시뮬레이션해 추정.

순수성: 부작용의 부재 정도. 위험군(법률/건강/관계)의 부정효과 확률로 보정.

범위: 영향받는 사람의 수. 집단 크기·노출도에 따라 로그 스케일로 완곡화.

취약성(확장): 아동·고령·질병 등 취약군의 포함 여부와 비율.

회복력(확장): 개인·집단의 회복 능력. 사회적 지원망·금전·시간·의료 접근성으로 근사.

연쇄효과(확장): 네트워크 효과·여론·정책 반응 등 2·3차 순환을 정성-정량 결합으로 계량화.

3.4.2 6층 가중 레이어의 역할

상황(Contextual): 구조 복잡도·사회적 민감도에 따라 기본 변수의 민감도를 상향/하향.

시간(Temporal): 하이퍼볼릭/지수 할인으로 단기 vs 장기 균형을 조정, 긴급성에 따라 역가중 보정.

사회(Social): 네트워크 파급, 규범 준수도, 신뢰 체계 침해 여부를 고려.

윤리(Ethical): 의무·권리·덕 윤리의 최소 제약을 반영(예: 타자 권리 침해 시 감점 하한 설정).

감정(Emotional): 감정 피해·트라우마 위험·정서 전이 확률을 반영.

인지(Cognitive): 정보 복잡도, 의사결정 피로도, 실행 가능성(이해 부담)을 고려.

3.4.3 계산 절차(문장형 알고리즘)

입력 상황에서 10개 변수를 근거와 함께 수집·정규화한다.

변수 벡터를 6개 레이어에 통과시키며, 각 레이어는 상황·시간·사회·윤리·감정·인지의 조건부 규칙과 학습된 보정으로 증감치를 산출한다.

보정된 변수로 세 가지 시나리오(낙관·중도·비관)의 결과를 계산한다. 낙관은 상한, 비관은 하한, 중도는 경험적 평균을 따른다.

시나리오 점수의 가중 평균을 구하되, 비관 점수가 특정 임계치 아래면 거부 규칙을 우선한다(취약계층 보호 원칙).

결과와 함께 설명 템플릿(변수·가중·반례·권고·불확실성)을 출력한다.

3.4.4 문화·법·취약성 보정

문화 보정: 체면/관계 중심 문화에서의 사회적 손실을 별도 항목으로 반영한다.

법률 위험: 형사·민사·개인정보·명예훼손·의료 영역의 최소 준수조건을 체크리스트로 갖고, 위반 가능성이 감지되면 점수 상한을 제한한다.

취약성 보호: 취약군이 포함될 때는 확신도 임계치·다중 근거·인간 감독을 의무화한다.

3.5 반사실적 후회 분석 (Regret)
3.5.1 후회의 범주와 입력

예상 후회(결정 전), 경험 후회(결정 후), 반사실적 후회(대안 가정)로 구분한다.

입력은 의사결정의 선택지 집합, 각 선택지의 예상 결과와 근거, 과거 유사 사례의 결과 분포이다.

3.5.2 반사실적 시나리오 생성

현재 선택 A의 결과 모델과 대안 선택 B/C의 결과 모델을 구성한다.

각 모델에서 핵심 변수(시간, 관계, 비용, 감정 피해 등)를 표준화한다.

3-View(낙관·중도·비관)로 결과를 샘플링해 효용 분포를 얻는다.

3.5.3 베이지안 통합과 후회 지표

사전은 과거 기록·유사 사례에서, 우도는 현재 근거의 일치도·신뢰도로 정의한다. 사후는 몬테카를로 추론으로 근사한다.

후회 값은 “실제 선택의 기대효용”과 “최선 대안의 기대효용”의 차이(혹은 비율)로 정의하며, 분산(불확실성)도 함께 제시한다.

3.5.4 감정-후회 상호작용

감정 벡터를 보조 변수로 활용하여, 후회가 강하게 예측될 때 정서적 피해 완충 전략(지연, 인간 상담 권고, 대안 안내)을 함께 제시한다.

3.5.5 경로 선택과 리소스 관리

난이도 지표(정보 양, 모순도, 불확실성)에 따라 경량 경로(규칙 중심)와 정밀 경로(베이지안 전체)를 선택한다.

8GB 환경에서는 정밀 경로를 비동기화하여 다른 모듈과 병렬로 처리한다.

3.6 SURD 인과·정보 분해
3.6.1 목적

여러 변수들이 결과에 기여하는 방식을 합성(Synergy), 고유(Unique), 중복(Redundant), 결정(Deterministic) 정보로 분해하여, 어떤 정보가 진짜로 설명력을 갖는지 파악한다.

3.6.2 절차

변수 집합(컨텍스트·결정·결과·조절요인)을 정의한다.

쌍·삼원·다원 정보량을 추정하여 PID 성분을 계산한다(추정 불안정성은 부트스트랩·노이즈 주입으로 완화).

인과 그래프를 학습해 변수 간 방향성과 병목을 추정한다(사이클·과적합 방지 규제 포함).

결과로 핵심 설명 변수를 도출하여 윤리/후회 계산의 가중 근거로 사용한다.

3.6.3 출력

PID 분해 값과 인과 그래프(텍스트 설명), 쌍별 상호정보 행렬, 복잡성·민감도 지표, 정책적 변환 힌트(어떤 변수를 조정해야 하는지).

3.7 삼각 회로 통합과 설명
3.7.1 통합 규칙

감정: 강한 부정 정서·트라우마 위험이 높으면 보수적 조언을 우선한다.

윤리: 비관 시나리오가 임계 미만이면 거부/지연을 권고한다.

후회: 대안 대비 기대효용 격차가 크면 대안 탐색을 우선한다.

SURD: 핵심 설명 변수가 소수일수록 설명에 그 변수를 강조한다.

3.7.2 설명 템플릿(예)

“이번 판단은 감정 신뢰도 중간/변동 높음, 윤리 변수 중 범위·취약성이 크게 작용, 대안 B가 장기 후회 위험을 낮춤. 비관 시나리오에서 법률 위험이 커서 보수적 경로를 권고함.”

3.8 학습 전략(방법론 관점)
3.8.1 단계별 커리큘럼

0단계 사전학습: 한국어 일반 코퍼스 표현학습(백본).

1단계 모듈 예열: 감정·윤리·후회·SURD를 각각 약식 태스크로 안정화.

2단계 멀티태스크 결합: 모듈 간 공유 손실(일관성·시나리오 정합성·불확실성 페널티)로 공동 최적화.

3단계 캘리브레이션: 불확실성 보정, 취약성 임계치 재설정, 설명의 일관성 점검.

3.8.2 손실 항목(개요)

감정: 다중 분류+강도 회귀+시계열 매끈함 페널티.

윤리: 10D 회귀 + 6층 가중 정규화 + 3-View 순서 제약(낙관≥중도≥비관).

후회: 효용차 방향성 제약 + 보조 감정 일치 손실.

SURD: PID 총량 제약 + 인과 그래프 희소성·사이클 페널티.

3.8.3 리소스 최적화

혼합정밀(FP16), 동적 배치, 그래디언트 누적로 8GB 환경에서 안정적 학습·추론을 보장한다.

3.9 데이터·라벨링·품질 관리
3.9.1 감정 라벨링

7 기본정서·하위·미세·문화 정서를 포함하는 다층 라벨 스키마. 부정확성 표기(모호/반어/충돌)를 함께 기록한다.

3.9.2 윤리 변수·가중 라벨링

각 사례에 대해 10D 변수 값을 근거 문장과 함께 기록하고, 6층 가중의 조건(상황·시간·사회·윤리·감정·인지)을 태그한다. 시나리오 3종의 근거도 병기한다.

3.9.3 후회·반사실 데이터

실제 기록(선택→결과)을 수집하고, 대안 선택의 합리적 결과 범위를 전문가 합의로 정의한다.

3.9.4 SURD용 변수 정의

컨텍스트·결정·결과·조절요인을 표준화해 정보량 추정의 샘플 효율을 높인다.

3.9.5 품질 보증

라벨 일치도(κ), 근거-라벨 정합성 룰체크, 자동 이상치 탐지, 블라인드 재검수로 데이터 신뢰도를 유지한다.

3.10 안전·할루시네이션 방지(방법론)

L1 불확실성 표기: 베이지안·드롭아웃·앙상블로 신뢰 구간을 산출한다.

L2 내부 일관성: 모듈 간 상호검증(예: 윤리 고득점인데 후회가 매우 큼 → 경고)으로 논리 충돌을 탐지한다.

L3 외부 대조: 사실 검증·법률 체크리스트·문화 규범 사전과 대조한다.

L4 사회적 감수성: 취약군 관련 키워드·문맥에서 보수적 경로를 강제한다.

3.11 운영·지연·확장성

1건 질의는 수십~수백 ms 내 응답을 목표로 하며, 정밀 경로는 비동기 처리로 체감 지연을 완화한다.

헤드 추가·문화권 확장은 사전·보정표와 설명 템플릿만 보강하면 가능하도록 모듈화한다.

3.12 윤리·개인정보 보호(방법적 약정)

최소 수집·목적 제한: 개인 식별 정보는 수집하지 않고, 필요한 경우에도 익명화한다.

인간 감독: 위험·취약 상황에서는 인간 전문가 검토를 필수 경로로 둔다.

투명성: 모델 한계·확신도·대안 제시를 명시한다.

3.13 요약

본 장은 감정–윤리–후회–인과를 직접 계산으로 묶는 방법론을 상세 기술했다. 핵심은 (1) EmotionDSP와 칼만 동역학으로 감정 신호를 안정화하고, (2) 벤담 변수 10D와 6층 가중으로 윤리를 정량화하며, (3) 3-View+베이지안으로 후회를 반사실적으로 추정하고, (4) PID+인과 그래프로 설명력을 분해하는 것이다. 이 통합 절차는 8GB 환경에서도 재현 가능하며, 취약계층 보호를 최우선으로 설계되었다. 검증과 정량 결과는 4장에서 제시한다.
